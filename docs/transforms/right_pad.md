##### Function: `right_pad`
```
  Pad `text` with the minimum number of copies of `pad` to reach `desired_length`.

Examples:

  right_pad('hello', 10, 'x')
  -- Result: "helloxxxxx"

  right_pad('hello', 10, 'xy')
  -- Result: "helloxyxyx"




```

###### Signatures
    text, number, text -> text
