##### Function: `+`
```
  Keep a number's sign


  Add two numbers together

Examples:

  +(2)
  -- Result: "2"

  +(-2)
  -- Result: "-2"


  2 + 2
  -- Result: "4"

  2.5 + 3
  -- Result: "5.5"




```

###### Signatures
    number -> number
    number, number -> number
