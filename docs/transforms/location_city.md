##### Function: `location_city`
```
  Extract the city from a location

Examples:

  make_location('3331 39th Ave', 'Vancouver', 'WA', '98818', to_point('POINT (42 42)')).city
  -- Result: "Vancouver"




```

###### Signatures
    location -> text
