##### Function: `location_address`
```
  Extract the address from a location

Examples:

  make_location('3331 39th Ave', 'Vancouver', 'WA', '98818', to_point('POINT (42 42)')).address
  -- Result: "3331 39th Ave"




```

###### Signatures
    location -> text
