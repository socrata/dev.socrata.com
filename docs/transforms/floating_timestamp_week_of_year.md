##### Function: `floating_timestamp_week_of_year`
```
  Extract the week from a calendar date as an integer between 0 and 51.

Examples:

  to_floating_timestamp('2017-12-13T12:34:55').week_of_year
  -- Result: "50"




```

###### Signatures
    calendar_date -> number
