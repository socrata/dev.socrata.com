##### Function: `left_pad`
```
  Pad `text` with the minimum number of copies of `pad` to reach `desired_length`.

Examples:

  left_pad('hello', 10, 'x')
  -- Result: "xxxxxhello"

  left_pad('hello', 10, 'xy')
  -- Result: "xyxyxhello"




```

###### Signatures
    text, number, text -> text
