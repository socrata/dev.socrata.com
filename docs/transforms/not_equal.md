##### Function: `!=`
```
  Return true if the left side does not equal the right

Examples:

  1 != 1
  -- Result: false

  1 != 2
  -- Result: true

  'a' != 'b'
  -- Result: true




```

###### Signatures
    a, a -> checkbox
