##### Function: `location_zip`
```
  Extract the zip from a location

Examples:

  make_location('3331 39th Ave', 'Vancouver', 'WA', '98818', to_point('POINT (42 42)')).zip
  -- Result: "98818"




```

###### Signatures
    location -> text
