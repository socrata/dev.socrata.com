Help!!!

So, it sounds like you’ve hit a snag and need a hand. Fortunately we have an amazing Support Team that can assist you, depending on what you’re looking to do!

If you’re an app developer...

If you’ve got questions about how to access Socrata data programmatically, or even just a general question about Socrata, you can submit a support ticket to our Support Team, and we’ll help you out!

If you’re a data publisher...

If you’re a data publisher and you need help setting up data publishing, or if you have publisher-specific questions, you can also submit a support ticket for our Support Team to have a look at. You can also reference our online knowledge base, which is chock-full of helpful articles about many of the most frequently requested publisher topics.

If you’ve found a documentation bug...

Our developer documentation is all managed through GitHub Pages. If you've found a bug, please feel free to file an issue on GitHub or follow our contribution guidelines and send us a pull request!
