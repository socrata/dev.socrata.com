Help!!!

So, it sounds like you’ve hit a snag and need a hand. Fortunately we have a number of options available for you to seek assistance, depending on what you’re looking to do. Find the closest match below and we’ll be there to help you out!

If you’re an app developer...

If you’ve got questions about how to access Socrata data programmatically, or even just a general questions about Socrata, you can post them to Stack Overflow using the soda or socrata tags and we’ll help you out!. Some of us also hang out in IRC during business hours in Seattle, if you've got a quick question.

If you’re a data publisher...

If you’re a data publisher and you need help setting up data publishing, or if you have publisher-specific questions, you can also submit a trouble ticket for our support team to have a look at. You can also reference our online knowledge base which is chock-full of helpful articles about many of the most frequently requested publisher topics.

If you’ve found a documentation bug...

Our developer documentation is all managed through Github Pages. If you've found a bug, please feel free to file an issue on Github or follow our contribution guidelines and send us a pull request!
